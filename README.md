### 🧪 Currently Tinkering With

The future of building software with AI is shifting fast — here's what I'm exploring right now:

- 🤖 **Agentic Engineering** — designing autonomous AI agents that can plan, reason, and execute tasks
- 🧠 **Context Engineering** — crafting the right context windows, memory, and retrieval strategies for LLMs
- 🎯 **Prompt Engineering** — building robust prompts and prompt chains for reliable AI outputs
- 🔧 **Harness Engineering** — building evals, tooling, and infrastructure around AI systems
- 🔌 **MCP (Model Context Protocol)** — connecting LLMs to external tools and data sources
- 📚 **RAG & Vector Databases** — retrieval-augmented generation pipelines for grounded AI responses
